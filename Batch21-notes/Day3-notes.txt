-> GitHub: (Central repository to store our code)

GitHub is a web based platform on top of git that lets you host your git repositories online.

It mainly used for

-> Collaborating with others
-> Storing the code in the web
-> working on open source projects
-> Providing features like issue tracking, pull request and CI/CD...etc


Commands:

1. git clone <Repo URL>
2. git add .
3. git commit -m "message"
4. git push origin <branch name>
