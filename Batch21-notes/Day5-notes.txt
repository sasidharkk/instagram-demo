-> Release branch -> Release the code to the end user/ customer

# Git stash:


git stash is a command used to temporarily save uncommitted changes(both staged and un staged) without committing them, so that we can switch branches or work on something else. then we can bring  those changes back later.

# git fork:

Copy entire someone's repository to your project (your own git hub account)

/ copy repositories from one user to another user
